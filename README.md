# rslv
Implements the rslv cli for resolving things.

```
rslv [OPTIONS] identifier

-t, --trace   Trace the resolution step to completion.
-m, --meta    Provide metadata about the identifier.
-f, --format  Request resolution of the specified media-type. Multiples applied in order of preference.
--formats     Present a list of media-type shortcuts and exit.
--schemes     Present a list of known identifier schemes.
```
